# Evals

Use these checklists before building, importing, activating, or expanding GTM automation loops.

Start with `gtm-loop-evals.md` for grounding, safety, build-readiness, and validation checks.

Minimum v1 gate:

- Workflow readiness passes.
- Privacy/security risks are documented.
- Human approval gates exist.
- Test payloads validate.
- Activation remains off until explicit approval.

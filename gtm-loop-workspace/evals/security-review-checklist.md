# Security Review Checklist

- [ ] No credentials in prompts, docs, payloads, or workflow JSON.
- [ ] Least-privilege scopes are documented.
- [ ] Sensitive data is redacted from logs.
- [ ] Writes require approval.
- [ ] External sends require approval.
- [ ] Workflow activation requires approval.
- [ ] Error messages do not expose secrets.
- [ ] Access assumptions are documented.

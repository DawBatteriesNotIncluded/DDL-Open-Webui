# Workflow Readiness Checklist

- [ ] Loop spec exists.
- [ ] Trigger is defined.
- [ ] Input schema is defined.
- [ ] Test payload exists.
- [ ] Credentials and scopes are listed.
- [ ] No secrets are hardcoded.
- [ ] Approval gate exists before side effects.
- [ ] Error handling exists.
- [ ] Idempotency exists for writes.
- [ ] Logging exists.
- [ ] Rollback plan exists.
- [ ] Activation approval is explicit.

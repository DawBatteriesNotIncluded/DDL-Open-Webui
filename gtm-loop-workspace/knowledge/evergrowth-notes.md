# Evergrowth Notes

## Role

Evergrowth can provide external account and buying signals.

## Signal Evaluation

Score each signal by:

- Source reliability.
- Recency.
- Company match confidence.
- ICP fit.
- Commercial relevance.
- Urgency.
- Evidence strength.

## Safe V1 Pattern

Signal -> account match -> score -> seller brief -> approval -> optional HubSpot task.

# Loop Engineering Notes

Loop engineering designs repeatable AI-assisted systems, not one-off prompts.

## Required Components

- Trigger.
- Context retrieval.
- Planner/agent task.
- Tool execution.
- Evaluation/check.
- State/memory/logging.
- Human approval where needed.
- Next action.
- Stop condition.

## Design Bias

Start with a safe, manual-review v1. Automate writes only after quality, test coverage, monitoring, and approval are proven.

# Open Questions

Cross-client questions that affect future agents. Keep tactical questions in `clients/<client-slug>/open-questions.md`.

| Question | Client | Status | Owner | Evidence Needed |
| --- | --- | --- | --- | --- |
| Which client is currently active? | Global | Unknown | User | Update `llm-wiki/current-client.md`. |

## Question Rules

- Ask only questions that change implementation, scope, risk, or handoff quality.
- Close questions with evidence, not memory.
- If a question is client-only, keep it in the client folder and link it here only when it affects shared patterns.

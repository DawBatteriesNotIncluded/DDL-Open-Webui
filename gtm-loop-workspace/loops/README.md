# Loops

Loop specs define repeatable AI-assisted systems with trigger, context retrieval, planner/agent task, tool execution, evaluation, state/logging, human approval, next action, and stop condition.

Use these files as Open WebUI Knowledge and as starting specs for n8n workflow design.

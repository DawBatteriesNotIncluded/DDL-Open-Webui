# Client Discovery Prompt

```text
Run a practical GTM automation discovery session.

Goal:
Understand enough client context to create or update clients/<client-slug>/.

Ask for:
- business objective;
- systems in scope;
- current manual workflow;
- existing automations;
- source of truth for each record type;
- approval gates;
- sensitive data;
- target outputs;
- known failure cases;
- deadline and owner;
- available evidence: docs, exports, screenshots, repos, workflow JSON, API docs.

Output:
- client context summary;
- system inventory;
- open questions;
- first workflow candidates;
- evidence needed next.
```
